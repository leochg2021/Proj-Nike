# 🏃‍♂️ Nike Air Zoom - Landing Page

Landing page estilosa e responsiva para promover o tênis **Nike Air Zoom**, criada com foco em design moderno, cores vivas e animações visuais.

<img src="https://github.com/leochg2021/Proj-Nike/blob/main/img/capaProj.png">

---

## ✨ Funcionalidades

- 🎨 Escolha de cores do modelo
- 🖼️ Imagem animada do tênis com sombra realista
- 💬 Texto institucional com ênfase no conforto
- 📱 Responsivo para desktop, tablet e mobile
- 💡 Fundo dinâmico com faixa preta diagonal estilizada
- 🔠 Tipografia moderna com fonte [Noto Sans](https://fonts.google.com/specimen/Noto+Sans)
